# Project Memory: noteweb_v2 (Laravel + Vite)

## Perintah Utama (Core Commands)
- **Development Setup:** `composer install && npm install`
- **Serve Application:** `php artisan serve`
- **Asset Compilation (Vite):** `npm run dev` / `npm run build`
- **Database Operations:** `php artisan migrate` / `php artisan db:seed`
- **Running Tests:** `php artisan test` atau `./vendor/bin/phpunit`

## Arsitektur & Teknologi
- **Framework:** Laravel (Standard MVC Structure)
- **Frontend Asset:** Vite
- **Database:** PostgreSQL (Neon.tech)
